Fraser Speirs - Blog - Misconceptions About iOS Multitasking

Let me wrap this up by giving you a quick summary: If someone tells you that all the apps in the multitasking bar are running, using up memory or sucking power, they are wrong. When you hit the home button, an app moves from Active to Background and quickly to the Suspended state where it no longer uses CPU time or drains power. An app may request an additional 10 minutes of Background running to complete a big task before becoming Suspended.
